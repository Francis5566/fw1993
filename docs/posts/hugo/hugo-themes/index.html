Hugo themes

Hugo 官方線上 themes

a. 直接 submodule

不考慮更改 themes 的 partial/layout/css 的話，直接照抄就好 如果不太更動 theme 或是內容比較一般性的話，可以直接 clone。

# submodule
cd ./themes
git submodule add https://github.com/avianto/hugo-kiera.git kiera

b. fork 再 submodule

如果需要更改一些比較客製化的版本的話請選擇 fork。

# fork
cd ./themes
git clone https://github.com/$你的Id/hugo-kiera.git kiera

修改 config.toml

theme = "kiera"
