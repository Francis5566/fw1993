Linux filesystem

Dec 28, 2019

/bin

  • 系統基礎的 Binary 檔案
  • 例如： ls, mv, rm, mkdir, rmdir 常用的執行檔
  • 內容和/usr/bin 基本相同，有時候會使用連結檔

/dev

  • 裝置相關的檔案或特殊的檔案
  • Unix 或 Linux 系統均把裝置當成是一個檔案來看待

/etc/

  • /etc 存放各種 Host/系統 設定檔
  • 裡面的檔案是 static 的
  • 不得存放 binary

/etc/rc.d

  • 記錄一些開關機過程中的 scripts 檔案， scripts 有點像是 DOS 下的批次檔
