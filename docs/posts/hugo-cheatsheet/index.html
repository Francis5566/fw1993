Hugo 使用

安裝

brew install hugo

第一次使用

hugo new site quickstart

測試

#!/bin/bash
cd quickstart
hugo
hugo server

部署

這邊有個小坑，content 寫好 markdown，指令下 hugo 不會自動覆蓋到最終的資料夾裡面，需要手動刪除再跑一次 hugo 。

rm -rf ./docs
hugo

新增文章

hugo new posts/my-first-post.md
