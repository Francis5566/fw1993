git cheatsheet

  • Dec 27, 2019
  • Categories: git
  • #git
  • One minute read

新增 submodule

git submodule add git@github.com:xxx dir_name

移除 submodule

git rm --cached [欲移除的目錄]
rm -rf [欲移除的目錄]

修改 .gitmodules

vim .gitmodules

修改 .git/config

vim .git/config

最後再 commit，改變整個 Repository。

$ git add .gitmodules
$ git commit -m "Remove a submodule"

安全起見再做個 sync

git submodule sync
